This is my first blog post, created using Claude Code!

## What is Claude Code?

Claude Code is an AI-powered coding assistant that helps with software engineering tasks. It can:

- Read and understand codebases
- Make code changes and improvements
- Set up new features like this blog
- Answer questions about code

## How This Blog Works

This blog is powered by Jekyll, a static site generator. To add a new post, I simply:

1. Create a new Markdown file in the `_posts` folder
2. Name it with the format `YYYY-MM-DD-title.md`
3. Add front matter (title, date, layout)
4. Write the content in Markdown
5. Commit and push to GitHub

GitHub Pages automatically rebuilds the site, and the new post appears!
